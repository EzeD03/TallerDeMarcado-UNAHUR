1. El DOCTYPE debe ir dentro del elemento <head> : FALSO
2. El atributo lang en <html lang="es"> le indica al navegador en qué idioma está escrito el contenido : VERDADERO
3. El elemento <title> define el título que aparece dentro de la página visible al usuario : FALSO
4. Un documento HTML puede tener más de un elemento <body> : FALSO
5. La etiqueta <meta charset="UTF-8"> es necesaria para que el navegador muestree correctamente tildes y la letra ñ : VERDADERO
6. El elemento <head> puede contener texto visible para el usuario : FALSO
7. <link rel="Stylesheet" href="estilos.css"> se usa para vincular una hoja de estilos CSS externa : VERDADERO
8. El elemento <script> solo puede ir al final del body, nunca en el <head> : FALSO
