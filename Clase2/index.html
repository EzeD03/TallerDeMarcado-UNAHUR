//DOCTYPE es una declaración que se utiliza para informar al navegador sobre la versión de HTML que se está utilizando en el documento. En este caso, se está utilizando HTML5, que es la versión más reciente y ampliamente compatible de HTML. La declaración es esencial para garantizar que el navegador interprete correctamente el contenido del documento y aplique las reglas de estilo adecuadas. Sin esta declaración, el navegador podría entrar en un modo de compatibilidad que podría afectar la apariencia y funcionalidad del sitio web. //El elemento es el elemento raíz de un documento HTML y contiene todos los demás elementos. El atributo lang="es" especifica el idioma del contenido del documento, en este caso, español. Esto es importante para la accesibilidad y para que los motores de búsqueda puedan entender el idioma del contenido. Además, algunos navegadores pueden utilizar esta información para aplicar reglas de estilo específicas para ese idioma. Document
